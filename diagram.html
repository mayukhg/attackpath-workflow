01 Discover The full attack path chain — from Internet Entry Point to Crown Jewel — surfaces in the attack paths table. User views CID, severity, hop count and risk score to prioritise which path to investigate.
02 Analyze User drills into the selected path to inspect asset type breakdown, shared resource warning banner, node hover metadata (with 🔗 in N paths badge), cross-path resource membership, Resource Detail Panel slide-over, privilege escalation steps, MITRE ATT&CK mapping, toxic combinations, and identity / lateral movement chain.
03 Remediate Prioritized fixes with ROI scoring — patch CVE-2024-1234, restrict IAM role, encrypt S3 bucket. Shared-resource actions show 🔗 Affects N paths cross-path blast radius. Each action shows ETA, Effort, and ROI. User marks actions Fixed, raises tickets, or suppresses with justification.
